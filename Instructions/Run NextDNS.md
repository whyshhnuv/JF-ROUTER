To Run NextDNS on your Router.

An account is recommended on [NextDNS](https://my.nextdns.io) webpage for this guide

1. Download NextDNS binary from Release section or compile it yourself for your router.

2. Store it in /flash folder of router using ftp or you may use 
`cd /flash`
then `wget --no-check-certificate` copy and paste the link of binary here.

3. Run [this script](https://github.com/Naitik1208/JF-ROUTER/blob/main/Scripts/nextdns.sh)
4. Please Note that the script has a column your ID there enter your NextDNS profile ID or NextDNS will not work with your filter list and act as normal server.
5. All Done!! Enjoy NextDNS.
